# Office Runner
### Latest Version: 0.2.9 (Alpha)
## About
**Office Runner** is a singleplayer 3D endless runner game currently in early development.

## Project Status
Working on **Office Runner** is currently paused, due to missing time and other issues, listed below.
**Office Runner** is still in early development.
## Gameplay
**Office Runner's** gameplay focuses on the players decisions and its impacts.
## Controls
- Dodge Obstacles: `A`, `D`
- Jump: `Space`
## How to test?
1. Click on the following [link](https://playcanvas.com/project/1466556/overview/office-runner)
2. Click the orange 'Editor' button.
3. In the editor, click the 'Launch' button, located in the right corner.
- Please take note of the fact that the game is currently facing many difficulties, so at the time you are reading this, it could be
facing massive issues (will be fixed soon).

## Tech Stack
- JavaScript
- HTML
- CSS
- PlayCanvas
